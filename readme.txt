Import database
1. Install Xampp dan buka xampp nya
2. Hidupkan Apache dan Mysql
3. Masuk ke localhost/phpmyadmin di web browser
4. Buat database dengan nama "hr_data"
5. Import file 'har_data.sql' 

Menjalankan program
1. Install Anaconda beserta environmentya (python 3.8)
2. Jalankan environmentnya 
3. Install libary nya dengan ketik "pip install -r requirements.txt"
4. Jalankan program dengan ketik "python app.py"
5. Salin url pada cmd "http://127.0.0.1:5000/"
6. Paste url pada web browser
